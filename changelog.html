Changelog - What's new?

  • 0.4.2
    • - Add a feature to make SO page wider

  • 0.4.1
    • - Add a feature to allow expanding vote counts even when you don't have the required 1000 reputation points. It's enabled by default. Thanks to Rob W (who wrote the original script) and thedemons (who added the feature in this extension) for making this possible.
    - Fix "Show indicator when there isn't any answer to the question" feature, accidentally broke in previous update

  • 0.4.0
    • - Update extension to use the new Manifest v3, made mandatory by Google, to prevent the extension from breaking in a few months
    - Sidebar: remove number of favorites information from the popover, since this information isn't available anymore

  • 0.3.5
    • - Update answer percent score styling, to match more with SO theme (as requested here)
    - Sidebar: fix information popover not showing up anymore with SO website changes
    - Remove "Hide the useless StackOverflow left sidebar" feature since it can now be done natively from SO user preferences (see here)

  • 0.3.4
    • - Fix sidebar answer count which was broken due to StackExchange DOM update
    - Increase min duration between two auto expand votes, because it seems SO has increased it to ~2sec

  • 0.3.3
    • - Fix broken extension due to favorites now being bookmarks
    - Change a bit sidebar color to fit better with the new SO dark mode

  • 0.3.2
    • - Fix extension which was (again) broken due to StackExchange DOM update
    - Change URL matching wildcard to allow running on subdomains like ru.stackoverflow.com

  • 0.3.1
    • - Fix extension which was broken due to StackExchange DOM update

  • 0.3.0 (This update is pretty huge!)
    • - The code has been totally refactored, improved, documented in preparation to open sourcing. You can now contribute on the extension by making PR on GitHub. You can also submit bugs reports or feature requests there ;)
    - Add a feature to automatically expand all comments (hidden behind "show X more comments") when you use CMD/Ctrl + F to search in page. This way you ensure that you are searching the whole page and are not missing any results.
    - Add a feature to hide "Hot Network Questions" & "Meta Posts" blocks from SO right sidebar to prevent distraction (as requested by @Mikey Stengel)
    - Settings page : sort features by categories to improve readability + settings are now saved automatically when you change an option (no need to click the Save button anymore)
    - Improve the "Auto expand votes count" feature to avoid reaching SO limit while navigating through answers with arrow keys

  • 0.2.2
    • - Add a feature to bring scrolling behavior to vote buttons, so that you still see them even on long posts. This feature is enabled by default and can be disabled in settings (but I'm pretty sure you will love it!).
    - Answers Sidebar : add the question vote number in the popup implemented in 0.2.0
    - Fix the "upvote question from Answer Sidebar" feature implemented in 0.2.1 that was broken (sorry about that)

  • 0.2.1
    • - Answers Sidebar : add the ability to upvote question right from the answers sidebar, so that you don't have to scroll to the top to do it (as suggested by @Géza Kiss)
    - Answers Sidebar : add an indicator to know if question is upvoted (as suggested by @Géza Kiss)
    - Create a separate Changelog page that will appear every time the extension is updated

  • 0.2.0
    • - Fix extension for StackOverflow website update
    - Answers Sidebar : add the ability to hover the number of answers cell to get additional information about the current page
    - Small bug fixes in Answer Sidebar

  • 0.1.4
    • - Add new Stack Exchange websites support

  • 0.1.3
    • - Automatically disable Auto expand votes count & Show answer score if you doesn't have enough reputation (the setting is not disabled in this settings page, to prevent you from being unable to use it when you'll have enough rep)

  • 0.1.2
    • - Add option to disable the new left sidebar that appears on every page (it will only disable it on questions/answers pages, and not in profile etc)

  • 0.1.1
    • - Improve settings page

  • 0.1.0
    • - Initial release
