Changelog - What's new ?

  • 0.3.2
    • - Fix extension which was (again) broken due to StackExchange DOM update
    - Change URL matching wildcard to allow running on subdomains like ru.stackoverflow.com

  • 0.3.1
    • - Fix extension which was broken due to StackExchange DOM update

  • 0.3.0 (This update is pretty huge !)
    • - The code has been totally refactored, improved, documented in preparation to open sourcing. You can now contribute on the extension by making PR on GitHub. You can also submit bugs reports or feature requests there ;)
    - Add a feature to automatically expand all comments (hidden behind "show X more comments") when you use CMD/Ctrl + F to search in page. This way you ensure that you are searching the whole page and are not missing any results.
    - Add a feature to hide "Hot Network Questions" & "Meta Posts" blocks from SO right sidebar to prevent distraction (as requested by @Mikey Stengel)
    - Settings page : sort features by categories to improve readability + settings are now saved automatically when you change an option (no need to click the Save button anymore)
    - Improve the "Auto expand votes count" feature to avoid reaching SO limit while navigating through answers with arrow keys

  • 0.2.2
    • - Add a feature to bring scrolling behavior to vote buttons, so that you still see them even on long posts. This feature is enabled by default and can be disabled in settings (but I'm pretty sure you will love it !).
    - Answers Sidebar : add the question vote number in the popup implemented in 0.2.0
    - Fix the "upvote question from Answer Sidebar" feature implemented in 0.2.1 that was broken (sorry about that)
